# Changelog

All notable changes to the **Turbo Approve** extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-04

### Added

- 🚀 Initial release
- ✅ Auto-approve for AI coding assistant actions
- 🔄 Auto-scroll for editors and terminals
- ⚡ Status bar toggle with visual feedback
- ⌨️ Keyboard shortcut `Ctrl+Shift+T` / `Cmd+Shift+T`
- ⚙️ Granular settings (master toggle, auto-scroll, auto-approve, delay, notifications)
- 📊 Status quick-pick showing current configuration
- 🔔 Non-intrusive status bar notifications
- 📋 Output channel logging for debugging
- 🛠️ CI/CD pipeline with GitHub Actions
